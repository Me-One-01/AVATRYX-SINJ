# Model Files

This folder contains required model parameter files, SMPL-related assets, and supporting data files used by the SINJ reconstruction pipeline.

These files are not included directly in the repository because of their large size.

## Download Model Files

Download all required model files from the Google Drive link below:

https://drive.google.com/drive/folders/1dluS8SSXLdYDVghl9_N7zls7b12xVmbg?usp=sharing

## Instructions

1. Download all files from the Google Drive folder.
2. Place the downloaded files inside this directory:

SINJ/model_files/

3. Keep the folder structure and filenames unchanged.

## Notes

* These files are required for proper body mesh generation and reconstruction.
* Missing files may cause runtime errors during inference.
* The files are intentionally excluded from GitHub for repository optimization.
