# Pretrained Models

This folder contains the pretrained weights required for running the SINJ (Single Image Neural Joint Reconstruction) pipeline.

Due to GitHub file size limitations, the pretrained model file are not included directly in this repository.

## Download Pretrained Models

Download all required pretrained model file from the Google Drive link below:

https://drive.google.com/drive/folders/1cHOISs4S1NYFXqEJMf2MtfU6LLIGE24Y?usp=drive_link

## Instructions

1. Download the file from the Google Drive folder.
2. Place the downloaded file inside this directory:

SINJ/pretrained_models/

3. Ensure the filename remain unchanged.

## Notes

* These pretrained weight are required for mesh reconstruction and pose estimation.
* Without these file, the backend inference pipeline will not run correctly.
* The file are excluded from GitHub to keep the repository lightweight and manageable.
